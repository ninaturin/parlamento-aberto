# Dashboard de Emendas Parlamentares (Teste no Lovable)
## Confira o projeto em: https://emendas-parlamentares.lovable.app/

Este repositório contém um protótipo de dashboard de Business Intelligence desenvolvido como **teste de construção no Lovable**.

## Objetivo

Explorar a viabilidade de estruturar um dashboard analítico a partir de dados públicos de transferências de emendas parlamentares, com foco em:

* Transparência
* Organização dos dados
* Exploração visual interativa

## Contexto do projeto

Este projeto foi desenvolvido exclusivamente como **prova de conceito (PoC)** no ambiente Lovable, com o objetivo de:

* Testar a ingestão e modelagem de dados
* Avaliar possibilidades de visualização
* Validar uma estrutura inicial de análise

Não se trata de um produto final nem de um sistema oficial.

## Dados utilizados

* Bases de emendas impositivas (2023–2025)
* Dados públicos de transferências do Tesouro Nacional

Os dados seguem o **critério de caixa**, ou seja, os valores representam pagamentos efetivos realizados.

## Limitações

* O dashboard considera apenas valores pagos (não inclui valores indicados ou empenhados)
* Pode haver revisões nos dados de origem
* Estrutura ainda simplificada para fins de teste

## Tecnologias

* Lovable (construção do dashboard)
* Modelagem dimensional (Star Schema)

## Status

Protótipo em fase inicial, desenvolvido para experimentação e aprendizado.
